庆祝本站累计唯一用户已接近 **700**（统计口径：本站后端生成的去敏 `userId`，总数会随时间变化）。

为了把接下来要做的功能做得更好，我们准备了一份 **用户问卷**（约 10–15 分钟），包含：
- 你最常用哪些功能？哪里用起来最别扭？
- 你希望优先加哪些能力（比如：榜单/公开档案、图片生成、数据展示、移动端体验等）？
- 你更在意稳定性、速度，还是细节体验？

## 参与方式

- 问卷链接：https://v.wjx.cn/vm/twyt7dF.aspx#
- 截止时间：**长期**

## 安全提示（重要）

- 问卷 **不需要任何登录凭证信息**，请勿填写 `SessionToken / API Token / PhigrosToken` 等敏感内容。
- 我们只会用于功能优先级与体验细节的统计分析，不会公开单份答卷内容。
