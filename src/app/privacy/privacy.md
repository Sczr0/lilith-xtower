#### 2. 我们如何收集和使用您的个人信息

**2.1 收集目的与范围**  
本服务为实现成绩查询、生成图片、排行榜与公开档案等功能，会根据您的操作与选择处理必要的信息。我们坚持“**最小必要**”原则，仅处理实现功能所必需的数据。

我们可能处理的信息主要包括以下几类：

- **（1）登录凭证信息：** 当您通过 TapTap 授权或联合查分 API 登录时，我们会接收您提供的 SessionToken、APIID、平台 ID 等（统称“登录凭证”），用于访问您的 Phigros 云存档。该凭证仅用于建立查询会话，不用于其他目的。  
- **（2）操作记录与请求元数据：** 当您访问或使用本服务时，我们与第三方基础设施/统计服务可能会处理访问时间、IP 地址、User-Agent（浏览器/设备信息）、页面 URL（含路径与查询参数）等信息，用于服务运行、安全防护、性能分析、统计与问题排查。  
  - 我们**不主动在自建后端持久化保存包含明文登录凭证（SessionToken、APIID 等）的日志**。  
  - 但您理解并同意，Cloudflare、Vercel、Umami 等第三方服务可能会基于其产品机制与合规要求记录访问日志，我们无法完全控制其日志留存与存储位置，具体以其隐私政策/服务条款为准。  
- **（3）生成内容相关数据：** 当您发起成绩查询、生成图片或其他内容时，我们会在服务器内临时处理相应的 Phigros 成绩数据，以便输出结果。该处理通常为“即用即弃”，不保存原始云存档文件。  
- **（4）排行榜与公开档案数据：** 为提供全站排名及公开档案查询服务，在您开启“公开设置/公开档案”相关功能期间，系统将同步并在我们的数据库中长期保存您的部分非敏感游戏数据，包括但不限于：**RKS（潜力值）、RKS 组成、Best Top 3、AP Top 3、成绩摘要**以及您设置的**用户别名**（如启用）。此类数据不包含账号凭证，无法用于登录您的游戏账号。  
- **（5）赞助者列表数据：** 若您访问本服务的“赞助者”页面，我们会通过爱发电（Afdian）开放 API 获取赞助者列表数据（如昵称、头像、赞助套餐名称、累计金额等）并展示。该数据来源与展示规则以爱发电平台的设置与政策为准。

**2.2 数据使用方式**

- **登录凭证：** 仅用于验证授权并访问您的 Phigros 云存档数据；我们不将登录凭证写入排行榜数据库。  
- **Phigros 云存档数据：** 仅在运行时被解析用于生成查询结果或图片，处理完成后即释放，不存储原始文件。  
- **操作记录与请求元数据：** 用于服务运行、安全分析、性能优化、统计与问题排查。我们不会将其作为账号体系使用，也不会以此直接识别您的真实身份。  
- **排行榜与公开档案数据：** 用于生成全站排名、公开档案展示、反作弊与服务运营分析；当您关闭公开设置后，我们将停止入库与同步，并将对外访问入口关闭（详见第 4、5 节）。  
- **赞助者列表数据：** 用于展示赞助者鸣谢与赞助信息，不用于推送营销或对您进行画像。

#### 3. 我们如何使用 Cookie 和同类技术

**3.1 本地存储（localStorage 等）**  
本服务会使用浏览器本地存储（如 localStorage）保存部分设置与状态，以提升使用体验，包括但不限于：

- **偏好设置：** 深色/浅色模式等纯前端交互设置。  
- **会话维持：** 在您的浏览器本地保存登录凭证与必要的登录状态，减少重复输入。  
- **协议同意状态：** 记录您是否已确认阅读并同意本服务的《用户协议》与《隐私协议》。  

本地存储不等同于强加密。**请勿在公共/不受信任设备上保存凭证**；您可通过“退出登录”或“清除浏览器数据”进行清除。

**3.2 Cookie 与第三方脚本**  
为保证本服务正常运行与统计分析，我们及第三方服务商可能会使用 Cookie 或同类技术。需要说明的是：本服务接入的 Umami 已配置为 **完全 cookie-less** 模式，不会在浏览器端写入 Cookie，也不会自动生成并持久化“匿名访客 ID”，统计主要通过服务端方式聚合匿名数据；Cloudflare、Vercel 等第三方服务是否使用 Cookie 及其留存规则，以其各自隐私政策/服务条款为准。

#### 4. 我们如何披露您的个人信息

**我们不会将您的个人信息出售给任何第三方。** 我们仅在以下情形中可能披露或共享必要信息：

**4.1 法定情形**  
在法律法规、法律程序、诉讼或政府主管部门强制性要求的情况下，我们可能会披露您的个人信息。

**4.2 外部处理（依赖服务）**  
本服务的部分功能依赖第三方服务（如 TapTap 接口或联合查分 API）。当您主动选择使用相关登录方式时，您的请求数据（包括登录凭证与必要的请求参数）将发送至对应服务提供方以获取数据。此过程为功能实现所必需，若您不同意，请勿使用相关功能。

**4.3 聚合与匿名化统计的公开**  
您同意开发者有权对与服务运营相关的数据进行统计分析，并以匿名化/聚合形式公开发布全站访问量、活跃用户数、查询热度、年度报告等统计结论。此类统计结论不包含可识别到特定个人的身份信息。

**4.4 公开档案与排行榜展示**  
您理解并同意，在您开启“公开设置/公开档案”相关功能期间，您的 RKS（潜力值）及在全站的排名情况可能在排行榜中公开展示。除非您主动设置了“别名”，否则该等数据将以匿名或脱敏形式展示（如隐藏部分 ID）。

当您关闭公开设置后，我们将停止将您的数据入库/同步，您的数据将从排行榜与对外查询入口中移除，其他用户将无法再通过对外接口访问您的公开档案；  

若您此前已同步过历史记录，**彻底删除历史记录需要管理员手动删除**。如您希望彻底删除，请按本协议第 10 条联系方式联系开发者，我们将在合理期限内处理。  

#### 5. 您在个人信息处理活动中的权利

鉴于本服务不提供账号注册功能，且登录凭证主要由您在浏览器本地保存，您的权利行使方式与传统注册制平台有所不同：

**5.1 查阅与复制权**  
您可以通过本服务前端页面直接查看您当前的查询结果与生成内容。

**5.2 更正与更新**  
您可以通过重新登录/重新同步、修改别名或调整公开设置来更新与排行榜/公开档案相关的信息。

**5.3 删除权（数据清除）**  
（1）**本地数据：** 对于保存在您浏览器本地的凭证与设置，您可以通过“退出登录/清除浏览器数据”进行清除。  
（2）**即时会话数据：** 对于我们在服务器内存中“即用即弃”处理的 SessionToken 与云存档内容，停止使用或会话结束后将被释放。  
（3）**排行榜/公开档案数据：** 您可先在“公开设置”中关闭公开功能以停止同步与对外访问；如您希望彻底删除历史记录，请通过本协议第 10 条联系方式联系开发者并提供必要信息以便定位，我们将尽合理努力在 **15 个工作日**内处理。  

**5.4 用户授权与撤回**  
当您授权我们访问您的 Phigros 云存档时，表示您同意我们按本隐私协议所述方式处理您的数据。您可随时通过更换 SessionToken、撤销授权或停止使用本服务来撤回授权。撤回不影响撤回前基于授权已进行的合法处理活动。

#### 6. 我们如何保护您的个人信息

我们采取业界常见的安全防护措施来保护您的个人信息：

**6.1 传输安全**  
本服务全站支持 **HTTPS**，登录凭证与查询数据在传输过程中会进行加密保护，降低被第三方截取的风险。

**6.2 数据处理安全**  
我们采用“即用即弃”的数据处理策略：您的核心敏感数据（登录凭证、云存档文件）仅在必要时存在于易失性内存中，不将未匿名化的登录凭证数据或完整存档文件写入我们的排行榜数据库。

**6.3 终端侧安全提示**  
登录凭证通常保存在您的浏览器本地。请您妥善保管个人设备与浏览器环境，避免在不受信任设备/网络环境下使用本服务。

#### 7. 我们如何处理儿童的个人信息

若您是 **14 周岁以下的未成年人**，在使用本服务前，应事先取得您家长或法定监护人的同意与监督。尽管本服务不存储用户年龄信息，一旦我们发现未在监护人同意的情况下处理了儿童的个人信息，我们会尽合理努力进行删除或去标识化处理。

#### 8. 第三方提供商及其服务

为实现本服务功能，我们可能会接入第三方 SDK、API 或基础设施服务。典型第三方服务包括但不限于：

- **TapTap 服务器：** 用于验证登录凭证有效性与获取原始存档数据。  
- **联合查分 API：** 若您选择此方式登录，相关数据交互将遵循其隐私政策与服务条款。  
- **Cloudflare：** 提供 CDN 加速、DDoS 防护与 Web Analytics/Insights 等能力，可能会处理并记录访问日志（如 IP、User-Agent、请求 URL 等），用于安全分析与统计。  
- **Umami：** 用于访问统计与性能分析。我们已启用 IP 匿名化，并配置为 **完全 cookie-less** 模式：不会在浏览器端写入 Cookie，也不会自动生成并持久化“匿名访客 ID”，统计主要通过服务端方式聚合匿名数据；但该服务仍可能采集页面访问相关信息（包括页面 URL 的路径与查询参数），具体以 Umami 的隐私政策与部署配置为准。  
- **Vercel：** 用于托管前端站点与无服务器函数/边缘运行时。托管平台可能会在合规与排障需要下保留运行日志与请求元数据。  
- **Vercel Analytics：** 用于站点访问统计与性能分析，可能会处理页面访问与设备/浏览器相关信息，具体以其隐私政策为准。  
- **爱发电（Afdian）：** 用于赞助相关能力。我们会通过其开放 API 获取赞助者列表信息并展示；数据来源与展示规则以爱发电平台设置与政策为准。  

我们建议您在使用前查阅上述第三方的隐私政策。

**9.3 合规声明**  
您使用本服务，即视为您理解并同意上述基于技术架构所需的跨境即时处理。我们将尽合理努力选择主流合规的基础设施与统计服务，并在其能力范围内保障数据安全。
