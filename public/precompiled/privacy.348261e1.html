2. 我们如何收集和使用您的个人信息

2.1 收集目的与范围
本服务为实现成绩查询、生成图片等功能，会根据您的操作与选择收集必要的个人信息。我们仅收集实现产品功能所必需的数据，不会超出合理范围。

我们收集的个人信息主要包括以下几类：

  • （1）登录凭证信息： 当您通过 TapTap 授权或联合查分 API 登录时，我们会接收您提供的 SessionToken、APIID 或平台 ID 等，用于访问您的 Phigros 云存档。该凭证仅用于建立查询会话，不会用于其他目的。
  • 2）操作记录与日志信息： 当您访问或使用本服务时，服务器会自动记录访问时间、IP 地址、浏览器类型等信息。为保护您的隐私，您的 IP 地址及登录凭证在用于统计分析时，将在服务器端进行哈希（Hash）或去标识化处理，我们不会存储明文的关联数据。 这些数据仅用于服务运行、全站数据统计、安全分析及问题排查。
  • （3）生成内容相关数据： 当您发起成绩查询、生成图片或其他内容时，我们会在服务器内临时处理相应的 Phigros 成绩数据，以便输出结果。
  • （4）排行榜与公开档案数据： 为提供全站排名及公开档案查询服务，当您登录本服务时，系统将自动提取并持久化存储您的部分非敏感游戏数据，包括但不限于：RKS（潜力值）、RKS 组成、Best Top 3 及 AP Top 3 成绩信息。若您启用“别名”功能，我们还将存储您设置的用户别名。

2.2 数据使用方式

  • 登录凭证： 仅用于验证授权并访问您的 Phigros 云存档数据。
  • Phigros 云存档数据： 仅在运行时被解析用于生成查询结果或图片，处理完成后即自动清除，不会存储原始文件。
  • 操作记录与日志： 用于服务运行，数据统计、安全分析、问题排查，不会与任何个人身份信息关联。

3. 我们如何使用 Cookie 和同类技术

3.1 Cookie 的使用
为保证本服务的正常运行，我们可能会在您的计算机或移动设备上存储名为 Cookie 的小数据文件。Cookie 通常包含标识符、站点名称以及一些号码和字符。
本服务使用 Cookie 主要用于以下目的：

  • 偏好设置： 记录您的显示模式（深色/浅色）或其他纯前端的交互设置。
  • 会话维持： 在本地暂存您的本地凭证，以免去重复输入的繁琐。

4. 我们如何披露您的个人信息

我们不会将您的个人信息出售给任何第三方。 我们仅在以下情况中，可能会公开披露您的个人信息：

4.1 法定情形
在法律法规、法律程序、诉讼或政府主管部门强制性要求的情况下，我们可能会公开披露您的个人信息。

4.2 外部处理（依赖服务）
本服务的部分功能依赖于第三方服务（即《用户协议》中提到的“联合查分 API”或 TapTap 接口）。当您主动选择使用相关登录方式时，您的请求数据（包括 Token）将直接发送至上述服务提供方以获取数据。此过程为功能实现所必需，若您不同意，请勿使用相关功能。

4.3 聚合与匿名化数据的公开
您同意开发者有权对用户数据库进行分析并对用户数据库进行商业或非商业利用（包括但不限于公开发布全站访问量、活跃用户数、查询热度，年度报告等统计数据）。
此类统计数据将经过匿名化或去标识化处理（如 Hash 处理），不包含任何可识别特定个人的身份信息，不属于法律意义上的“个人信息”。

4.4 公开档案与排行榜展示
您理解并同意，作为本服务核心功能的一部分，您的 RKS（潜力值）及在全站的排名情况将默认参与统计并可能在排行榜中公开展示。
除非您主动设置了“别名”，否则该等数据将以匿名或脱敏形式（如隐藏部分 ID）展示。

您可以通过“功能菜单 - 公开设置”自主选择是否允许他人通过别名查询您的详细档案（如 Best Top 3、AP Top 3 等）。该等信息的公开范围完全由您控制，您开启相关选项即视为授权我们向其他用户展示该等数据。

5. 您在个人信息处理活动中的权利

鉴于本服务不提供账号注册功能，且不在本服务服务器持久化存储您的个人身份信息，您的权利行使方式与传统注册制平台有所不同：

5.1 查阅与复制权
您可以通过本服务的前端页面直接查看您当前的查询结果和生成的图片。

5.2 删除权（数据清除）
由于我们不在服务器硬盘或数据库中保存您的云存档数据或登录凭证（除非您使用浏览器端的本地存储），“退出页面”或“清除浏览器缓存”即等同于行使删除权。一旦您关闭页面，服务器内存中的相关数据将即刻释放。

5.3 用户授权与撤回
当您授权我们访问您的 Phigros 云存档时，表示您同意我们按本隐私协议所述方式处理您的数据。
您可随时通过更换 SessionToken、撤销授权或停止使用本服务来撤回授权。
撤回不影响撤回前基于授权已进行的合法处理活动。

6. 我们如何保护您的个人信息

我们采取业界标准的安全防护措施来保护您的个人信息：

6.1 传输安全
本服务全站支持 HTTPS 安全协议，您的 SessionToken 和查询数据在传输过程中均经过加密处理，防止被第三方截取。

6.2 数据处理安全
我们采用“即用即弃”的数据处理策略。您的核心敏感数据（SessionToken、存档文件）仅在本服务服务器的易失性内存中存在，不将未匿名化的登录凭证数据或完整存档数据写入任何永久存储介质，从根本上降低数据泄露风险。

7. 我们如何处理儿童的个人信息

若您是 14 周岁以下的未成年人，在使用本服务前，应事先取得您家长或法定监护人的书面同意。
尽管本服务不存储用户年龄信息，一旦我们发现未在监护人同意的情况下收集了儿童的个人信息，我们会设法尽快删除相关数据。

8. 第三方提供商及其服务

为实现本服务功能，我们可能会接入第三方 SDK 或 API。目前涉及的第三方服务如下：

  • TapTap 服务器： 用于验证 Token 有效性，获取原始存档数据。
  • 联合查分 API： 若您选择此方式登录，相关数据交互将遵循其隐私政策。
  • Cloudflare： 提供 CDN 加速与 DDoS 防护，可能会收集您的 IP 地址用于安全分析。
  • 统计服务（Cloudflare，Umami）： 仅收集匿名化的访问行为数据。
  • Vercel： 用于托管服务及无服务器函数（Serverless Functions）。

我们建议您在授权前查阅上述第三方的隐私政策。

9. 您的个人信息如何在全球范围转移

9.1 数据存储与持久化
（1）排行榜数据： 为实现排行榜功能，您的排名数据（RKS、成绩摘要、别名等）将被存储于我们的数据库中。此类数据不包含您的账号凭证，无法用于登录您的游戏账号。

9.2 数据处理位置
鉴于本服务使用 Vercel 提供的无服务器架构及 Cloudflare 提供的 CDN 网络，您的请求数据（含 IP 地址及传输中的加密凭证）可能会被路由至全球各地的边缘节点（包括但不限于中国香港、美国、新加坡等）进行即时处理。
此种数据传输仅为实现服务功能（如加速访问、API 响应）所必需。

9.3 合规声明
您使用本服务，即视为您理解并同意上述基于技术架构所需的跨境数据传输（仅限瞬时处理，非持久存储）。我们将以个人开发者的身份尽合理努力，要求上述第三方服务商保障数据安全。
